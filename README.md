AI agent and skill configurations, adapted from various sources. Including but not limited to:

- https://github.com/toddmaustin/research-prompts

Install
-------

Clone and run setup.sh to point all files and configuration directories to your
clone of the repository (can be run as many times as you want, in case you move
your clone around):

    git clone https://github.com/llvilanova/config-ai.git ~/config-ai
    ~/config-ai/setup.sh
