## What I do

- Extract text from a True PDF document (i.e., typed text is available, rather than rasterized)
  - Example tools: pdftotext, mutool, pypdf
- Extract images from a PDF document
  - Example tools: pdfimages, mutool, convert, pypdf
- Establish which tools are available for the desired extraction, and select the best one

## When to use me

Use this when you need to extract text or images from PDF files.
